## What Alt Maya Blender Ducktape Does

Alt Maya Blender Ducktape is a Blender plugin for anyone who wants a more familiar viewport navigation style without abandoning Blender's native keymaps. Press and hold Alt, and the add-on shifts into an industry-compatible navigation mode so orbiting behaves more like Maya.

It is a practical fit for artists and technical users who split time between Blender and other 3D packages like Maya, Houdini, Unreal Engine, Unity, Cinema 4D, Modo, Substance Painter, Substance Designer, Fusion, Mari, Clarisse iFX, and Nuke. The intent is to smooth out navigation across tools while leaving the default Blender experience available for tutorials and everyday production work.

## Key Features

- Uses an industry-compatible keymap only while Alt is held
- Reverts to Blender's default keymaps as soon as Alt is released
- Tweaks orbit behavior to feel closer to Maya-style navigation
- Makes it easier to switch between Blender and other 3D applications
- Keeps Blender's normal workflow intact for training and tutorials
- Offers a navigation pattern that feels natural to users from major DCC tools
- Targets viewport orbit and general navigation behavior specifically
- Lightweight add-on design that coexists with Blender's built-in input system

## Usage

Once the add-on is enabled, hold Alt while working in the viewport to temporarily switch into the industry-compatible navigation behavior.

Typical workflow:

- Open a 3D view in Blender
- Hold Alt
- Orbit, pan, or navigate as you normally would with a Maya-like setup
- Release Alt to return to Blender's default controls

This helps you stay productive in Blender without having to rebuild your muscle memory for every scene or task.

## FAQ

### Does this replace Blender's normal controls?
No. The plugin keeps Blender's default keymaps active whenever Alt is not being used.

### When does the Maya-like behavior activate?
It turns on while Alt is held, then switches back to the usual Blender setup afterward.

### Who is this for?
It is useful for Blender users who are accustomed to navigation in Maya or other major 3D applications with similar orbit workflows.

### Where are updates posted?
Use the repository and the download link at the top of this README for the latest build and project changes.

### What should I do if the navigation does not change?
Make sure the add-on is enabled in Blender, then verify your keymap settings and restart Blender if needed.
